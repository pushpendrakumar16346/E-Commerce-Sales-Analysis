# E-Commerce-Sales-Analysis
End-to-end data analysis project using SQL, Excel and Power BI.

Ecommerce Sales Dashboard (Power BI)

Project Overview
This project is a Power BI dashboard that analyzes ecommerce sales performance.
It provides insights into sales, profit, quantity, and regional performance.

Key Metrics,
YTD Sales,
YTD Profit,
YTD Quantity,
Profit Margin,
Dashboard Features,
Sales by Category,
Top 5 Products by Sales,
Bottom 5 Products by Sales,
Sales by Region,
Sales by Shipping Type,
Sales by State Map

Tools Used
Power BI,
Excel,
DAX
Data Visualization
Dataset

The dataset contains ecommerce sales data including:
Product category
Sales,
Profit,
Quantity,
Region,
Shipping type,
